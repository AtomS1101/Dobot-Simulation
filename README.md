# Dobot-Simulation
This project simulates a Dobot Magician using Python's turtle graphics.

## Requirements

- Python 3.x
- Turtle

## How to use?
You can import a simulator as a library as follows:

```python
from Simulator.robot import Robot, PTPMode
dType = Robot()
```

You can use almost all of the methods and functions supported by the DobotDllType library.
Note that some methods do nothing even if they are callable.　

`simulateTest.py` is a simple example that demonstrates three moving modes.
